# Social Statistics

## What is this?
Social Statistics is a responsive website about the effects of social media and healthier digital habits. It includes sections about why social media matters, healthier ways to use it, and key project facts.

## Why does it exist?
This project exists to help people understand both the benefits and drawbacks of social media use. Its goal is to promote more intentional, balanced, and healthy digital habits.

## What tools did I use?
- HTML for the page structure.
- CSS for layout, styling, responsiveness, and dark/light theme support.
- JavaScript for the theme toggle and local storage behavior.
- GitHub Pages or another hosting platform for deployment.
- I used AI tools to help draft and organize the content.
